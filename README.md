## Overview

### Requirements
* Any Operating System (ie. MacOS X, Linux, Windows)
* Flutter SDK (3.10.1 or superior)

### How to Run
* Inside root directory, run ```flutter pub get```
* After, run ```flutter run```

### Methodologies

* Clean Architecture
* Object Oriented Programming
* Reactive Programming

### Libraries and Tools

* Flutter
* Bloc
* Intl
